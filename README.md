Sales & Customer Insights Dashboard (Power BI)
This project presents an interactive Sales & Customer Insights Dashboard built in Power BI, designed to analyze business performance across sales, profitability, regions, products, and customer segments. It leverages DAX, Power Query, and a multi-table data model to generate actionable insights for data-driven decision-making.
